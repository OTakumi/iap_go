# IAP_Go

## IAP(Identity-Aware Proxy)

 Identity-Aware Proxyとは、ユーザーとアプリケーションの間に入って、通信を仲介するプロキシです。

- 参考情報
  - [アイデンティティ認識型プロキシ(Identity-Aware Proxy：IAP)](https://www.nec-solutioninnovators.co.jp/ss/insider/security-words/50.html)
  - [Identity-Aware Proxy](https://cloud.google.com/security/products/iap)

このプロジェクトでは、IAPをGoで自作し、その仕組みを理解する。
